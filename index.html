Demo for the DCI Typescript linter library eslint-plugin-dci-lint.

A tutorial is available at https://blog.encodeart.dev/dci-tutorial-for-typescript-part-1

Stackblitz cannot lint the DCI code in realtime, so the error that has sneaked into MoneyTransfer.ts is resulting in a faulty amount being deducted from the source account.

If you open a terminal, by clicking the "+" button in the terminal panel below and typing in eslint . you can see the error, and fix it.

When you use VS Code with the ESLint extension, it gives you linting as you type, and by including ESLint into your build process, it will make all Contexts adhere to DCI conventions and rules.
